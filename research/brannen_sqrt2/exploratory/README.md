# Exploratory `r6_c*` scripts — higher-order α coefficients

These 60 scripts belong to the "deep Brannen programme" (`r6_c*`):
higher perturbative orders of `α_n` beyond `α_3`, Green's-function
studies, Richardson extrapolations, PSLQ basis extensions, and various
diagnostics (quadosc error, Padé convergence, log-singularity tests).

They are **not** required to reproduce the four P4 closure results in
the paper (α₂, α₃, quark/neutrino Koide). The main-directory scripts
(`r6_c1_*`, `r6_c11`, `r6_c15`, `r6_c17`, `r6_c18`, `r6_koide_*`,
`r6_tau_constraint`) are the minimal set cited in §5 of
`paper/tgp_leptons.tex`.

Kept here for auditability of the full numerical workshop that preceded
the paper's final PSLQ statement (Prop. 7): "no relation involving
{1, ln 2, ln 3, π, π², Li₂(1/3), Cl₂(π/3), ...} with integer coefficients
bounded by 10¹⁴ matches TGP_Pcos at 40-digit precision." The exploratory
scripts document the steps that led to this negative result.

See `../README.md` for the full script index in the source working
repository.
